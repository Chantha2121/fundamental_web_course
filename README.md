# របៀបដំឡើង VS Code

## 1️⃣ Download
ចូល [code.visualstudio.com](https://code.visualstudio.com) ចុច Download (កម្មវិធីនឹងជ្រើសរើស OS របស់អ្នកដោយស្វ័យប្រវត្តិ)

---

## 2️. ដំឡើងតាម OS

### 🪟 Windows
1. បើក `.exe` file ដែលបាន download
2. ចុច Next រហូតដល់ចប់
3. **សំខាន់**: ត្រូវធីក ✓ **"Add to PATH"** និង **"Add 'Open with Code' action"**

### 🍏 macOS
1. បើក `.zip` file ដែលបាន download
2. អូស `Visual Studio Code.app` ទៅក្នុង `Applications` folder
3. (Optional) ចង់បើកពី Terminal:
   - បើក VS Code → ចុច `Cmd+Shift+P`
   - វាយ `shell command` → ជ្រើស `Install 'code' command in PATH`

### 🐧 Linux (Ubuntu/Debian)
1. Download `.deb` file
2. បើក Terminal ហើយវាយ៖
```bash
sudo apt install ./[filename].deb
```
(ជំនួស `[filename]` ជាមួយឈ្មោះ file របស់អ្នក)

---

## 3️. ក្រោយដំឡើងរួច

### 🔌 ដំឡើង Extensions សំខាន់ៗ
1. ចុច `Ctrl+Shift+X` (Windows/Linux) ឬ `Cmd+Shift+X` (macOS)
2. Search រក extension ខាងក្រោម រួចចុច Install:
   - **Prettier** (រៀបកូដឲ្យស្អាត)
   - **Live Server** (បើក website ផ្ទាល់)
   - **Khmer** (បើចង់បានភាសាខ្មែរ)
